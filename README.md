# Documentação do Projeto

## Instruções para Executar o Projeto

Este projeto contém notebooks Jupyter para a disciplina de Estatística Computacional.

### Pré-requisitos

- Python 3.8 ou superior
- [Jupyter Notebook](https://jupyter.org/install) ou [JupyterLab](https://jupyterlab.readthedocs.io/en/stable/)
- Recomenda-se o uso de um ambiente virtual (venv, conda, etc.)

### Passos para Execução

1. **Clone ou baixe o repositório**

   Navegue até a pasta do projeto no seu computador.

2. **Crie e ative um ambiente virtual (opcional, mas recomendado)**

   ```bash
   python -m venv venv
   .\venv\Scripts\activate  # Windows
   source venv/bin/activate   # Linux/Mac
   ```

3. **Instale o Jupyter e dependências**

   ```bash
   pip install notebook
   # Instale outras bibliotecas necessárias conforme solicitado nos notebooks
   ```

4. **Execute o Jupyter Notebook**

   ```bash
   jupyter notebook
   ```
   Ou, para JupyterLab:
   ```bash
   jupyter lab
   ```

5. **Abra os notebooks**

   Navegue até a pasta `comp_stats_class_1/` e abra os arquivos desejados:
   - `distribuicoes_e_periodos.ipynb`
   - `stats_comp_class_1.ipynb`

6. **Execute as células**

   Siga as instruções dentro dos próprios notebooks para executar as células de código.

### Observações
- Caso algum pacote esteja faltando, instale-o usando `pip install nome_do_pacote`.
- Para dúvidas ou problemas, consulte o README ou entre em contato com o responsável pelo projeto.
